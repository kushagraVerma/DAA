Decomposition of a Polygon into Convex Polygons

Abstract

We have implemented an algorithm to decompose a given (possibly non-convex) polygon into constituent disjoint convex polygons.

This algorithm has two steps: decomposition and merging.
  1. Decomposition finds constituent disjoint convex polygons, that can possibly be merged to give bigger convex polygons.
  2. Merging joins such smaller polygons by verifying that the angle at the common edge is not reflex.

We have also created a visualiser which can be used to generate inputs and verify outputs.

Optimizations

For the merging process, we do not need to enforce the checks given in the original algorithm because the Doubly Connected Edged List takes care of it through its internal structure. The only check we need to perform as part of the merging process is whether the vertices at the ends of the common edge of the polygons to be merged, are reflex.
