Ford-Fulkerson Algorithm

This is a well established algorithm for determining the maximum flow problem to a flow network.The max flow problem determins the maximum flow that can be sent from the source to sink in a network directed weighted graph. Along with the implementation of this algorithm it also aims to find the minimum s-t (source-sink) cut that is deducible from the max flow problem. This algorithm is also extened towards solving the Bipartite matching problem.

Line-Fitting Problem

The line fitting problem is a fundamental task in computational geometry and statistical analysis, with numerous applications across various fields. It involves finding the best-fitting line that approximates a set of data points in the most optimal manner. This abstract provides an overview of the line fitting problem, highlighting its significance, algorithms, and applications.
